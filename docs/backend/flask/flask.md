## Setup

### Linux

- Install Python3
- Install pip3
- Install virtualenv
- Create Virtual Environment along with python version to use ```virtualenv env -p python3.7```
- Activate Virtual Environment ```source env/bin/activate```
- Install Flask ```pip3 install flask```
- To Deactivate use ```deactivate```
